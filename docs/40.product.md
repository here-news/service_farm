## Part 1: Homepage â€” The Living News Ecosystem

**Traditional news homepages are graveyards.** A list of headlines sorted by recency.

**Our homepage is an observatory.** Users look into a living ecosystem where:
- Events have **mass** (importance from coverage depth)
- Events have **state** (ACTIVE/STABLE - are they breathing?)
- Events have **tension** (contradictions = unresolved debates)
- Events have **needs** (coverage gaps, unanswered questions)

### Visual Mapping

| State | Visual | Meaning |
|-------|--------|---------|
| ACTIVE | Pulsing, bright | Claims arriving, event breathing |
| WARM | Subtle glow | Recent activity, settling |
| STABLE | Solid, calm | Coherent, few contradictions |
| DORMANT | Faded | No recent activity |

**Tension â†’ Color:**
- Low (0.0-0.1): Blue/Green - Consensus
- Medium (0.1-0.3): Yellow - Some debate
- High (0.3+): Red/Orange - Active contradictions
